File Structure

Findeas follows a simple and easy to customize file structure. Here is the sample for your reference:

  • node_modules npm installs dependencies
  • src source files
    • assets assets files
      • css
      • icon
      • img
      • js
    • docs documentation file
    • php php files
    • scss scss files
    • *.html html document
  • .gitignore hide untracked files from git
  • gulpfile.js build tasks for development
  • LICENSE.txt licenses information
  • package-lock.json dependencies version tree
  • package-lock.json list of dependencies
